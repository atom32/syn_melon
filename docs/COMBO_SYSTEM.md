# 连击系统 (Combo System)

## 概述

连击系统为游戏增加了深度和刺激感。当玩家在短时间内连续合成水果时，会触发连击，获得分数加成。

## 机制

### 时间窗口
- **判定时间**: 1.5 秒
- **衰减时间**: 0.5 秒（连击窗口结束后延迟重置）

### 连击乘数

| 连击次数 | 分数乘数 | 文字颜色 |
|---------|---------|---------|
| 1次     | x1.0    | 白色    |
| 2次     | x1.5    | 绿色    |
| 3次     | x2.0    | 青色    |
| 4次     | x2.5    | 紫色    |
| 5次+    | x3.0    | 金色    |

## 视觉反馈

### 连击提示
- 屏幕中央显示 "Nx COMBO!"（N为连击次数）
- 下方显示当前乘数 "xM"（M为乘数）
- 使用 Tween 动画实现闪烁、放大效果
- 不同连击数使用不同颜色

### 浮动分数
- 显示 "+分数 (x乘数)" 格式
- 有连击时使用金黄色/橙红色
- 放大效果更明显

## 技术实现

### 核心文件

1. **ComboManager.gd** (autoload单例)
   - 管理连击计数和计时器
   - 计算连击乘数
   - 发射连击信号

2. **ComboDisplay.tscn/gd**
   - 屏幕中央的连击显示
   - Tween动画效果
   - 根据连击数改变颜色

3. **修改的文件**
   - `GameManager.gd`: 集成连击系统到分数计算
   - `Main.gd`: 连接连击信号到UI显示
   - `Fruit.gd`: 传递乘数到特效系统
   - `EffectManager.gd`: 支持乘数参数
   - `FloatingScore.gd`: 显示乘数信息

### 使用方法

```gdscript
# 触发连击（自动在合成时调用）
var combo_mgr = get_node("/root/ComboManager")
var multiplier = combo_mgr.trigger_merge(position)

# 获取连击信息
var info = combo_mgr.get_combo_info()
print("连击数: %d, 乘数: %.1f" % [info.count, info.multiplier])

# 重置连击（游戏重新开始时）
combo_mgr.reset()
```

## 游戏策略提示

- 玩家可以通过快速合成来维持连击
- 高连击数（5+）提供 x3.0 乘数，可以大幅提升得分
- 适合在多个低级水果堆叠后一起合成
- 注意1.5秒的时间窗口限制
