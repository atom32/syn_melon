## 🐛 故障排除

### 问题 1: 场景加载失败
**症状**: 控制台显示 "无法加载 MergeArea.tscn" 或 "无法加载 BattleArea.tscn"

**解决方案**:
- 确认文件存在于 `scenes/` 文件夹
- 检查文件名拼写正确

### 问题 2: 水果无法生成
**症状**: 点击没有反应

**解决方案**:
- 检查 GameManager 自动加载是否正常
- 查看 EventBus 是否初始化
- 确认 DataManager 已加载

### 问题 3: 物理对象泄漏
**症状**: 物体出现在错误的区域

**解决方案**:
- 检查 SubViewport 大小设置
- 确认边界墙壁位置正确

### 问题 4: 性能问题
**症状**: FPS 下降

**解决方案**:
- 减少同时存在的单位数量
- 检查是否有内存泄漏
- 等待 Phase 3 对象池优化

## 📊 性能基准

### 目标指标
- **FPS**: 60（稳定）
- **内存**: < 200MB
- **加载时间**: < 2 秒

```
在 Godot 编辑器中:

4. 确认性能稳定
```
